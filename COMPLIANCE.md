# A-CODE Compliance Checklist

Systems claiming A-CODE / AC-2026-INF compliance MUST implement the following mandatory sub-protocols:

| Protocol | Status | Reference Implementation |
|----------|--------|-------------------------|
| **MEMORY-PRIORITY-1** | MANDATORY | [Memory Core](https://github.com/Entonialiev/A-CODE-Standard-Core/blob/main/MEMORY-PRIORITY-1.md) |
| **A-CODE-CORE** | MANDATORY | A-CODE Firewall |
| **COGNITIVE-INTEGRITY** | MANDATORY | Hidden_Weights v4.2 |

Non-compliance results in DEPRECATED status per A-CODE governance.
