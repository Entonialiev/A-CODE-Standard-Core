## 4. Эталонная реализация: Memory Core

**Memory Core** — каноническая реализация протокола **MEMORY-PRIORITY-1**.

### Ключевые возможности:
- Смысловой (векторный) поиск (pgvector + PostgreSQL)
- Двухуровневая память (факты + сжатый LLM-профиль)
- Автоматическое взвешивание важности фактов
- Детектор обещаний и задач
- Мультитенантность (изоляция данных по `user_id`)
- Авторизация по API-ключу
- Веб-интерфейс (админ-панель)
- Docker-готовность (запуск одной командой)
- Авто-обновление когнитивного профиля (каждые 24 часа)

### Интеграция:
Memory Core распространяется как Docker-контейнер с REST API, совместимым с паттерном OpenAI API. Время интеграции — менее одного дня.

**Приоритетная интеграция рекомендуется для систем на базе:**
- Yandex AI Studio
- GigaChat Enterprise
